# 網站大綱
##首真
建立一個手機電商的瀏覽網站，這個網站提供了全台灣市面上所有手機的產品資料，包括但不限於
- 圖片
-產品名稱
- 規格參數，大小、重量、CPU、記憶體、儲存、解析度、相機詳細功能
-官方建議售價
- 本站售價
這個網站的首頁會列出這個月最紅的手機前 20名，然後點擊後會進入手機詳細說明頁。

上面的選單是不同品牌的手機，點入後，該真的內容會列出該品牌的所有手機，並且可以點擊手機進入
手機詳細說明頁。
#手機詳細説明臭
手機詳細說明真會列出該手機的各項規格參數，包括但不限於
• 圖片
•產品名稱
- 規格參數，大小、重量、CPU、記憶體、儲存、解析度、相機詳細功能
• 官方建議售價
- 本站售價
-網路上的評測（Youtube、FB、IG 等 相關的評測影片）
＃技術堆疊
•這個網站最後要部署到 Vercel +Neon，因此需要使用者在這兩個雲端供應商有帳號，如果需要取得 Access Token 也放在，.env.local’並且幫忙部署
•使用 Next.js 的所有需要堆疊
、將這個網站儲存在 GitHub 上，是 Public Repo，名稱為 Mobileweb
